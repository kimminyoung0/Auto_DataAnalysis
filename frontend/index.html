    분석 기록

      Auto Data Analysis

      날짜 필터

      날짜/시간 컬럼이 없습니다.

      데이터는 df · 예시를 클릭하면 입력창에 넣을 수 있습니다.

      • df — 전체 데이터
      • df.head(10) — 앞 10행만
      • df.tail(20) — 뒤 20행만
      • df.iloc[0:100] — 0~99행 (슬라이스)
      • df.iloc[[0, 10, 20]] — 특정 행만
      • df[['열이름1', '열이름2']] — 특정 열만
      • df[df['열이름'] == '값'] — 조건으로 행 필터
      • df.query('열이름 > 0') — query로 필터
      • df.sort_values('열이름') — 열 기준 정렬
      • df.groupby('MCCODE').head(3) — 그룹별 앞 3행

      적용: df

      업로드한 데이터의 변수 유형·기술통계를 생성합니다.
      위 「자동분석」 버튼을 눌러 실행하세요.

      데이터 품질 (결측/0값)

      수치형 컬럼의 결측(NaN)과 0값 비율을 확인합니다. 범주형 컬럼으로 그룹바이하여 비교할 수도 있습니다.

      최대 4개까지 선택 가능 · 기본 2개 선택

      비워두면 전체 그룹으로 분석합니다. (그룹 값 목록은 분석 후 자동 생성됩니다.)

      히트맵

      상세 표

      범주형 포함관계(그룹 카운트)

      범주형 컬럼을 1~10개 선택해 그룹바이 카운트를 만들고, 빈도표(count)로 포함관계를 확인합니다.
